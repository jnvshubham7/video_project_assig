# Video Upload, Sensitivity Processing, and Streaming App

This repository implements a full-stack video platform (video upload, processing, sensitivity analysis, streaming).

Structure:

- `/backend` - Node.js + Express + MongoDB backend
- `/frontend` - React + Vite frontend

Documentation: see `/docs` for Installation, API, User Manual, Architecture, Assumptions, Deployment, and Testing guides.
